title: Object API

The `Porffor.object` API provides low-level operations for object manipulation, property access, and object state management.

## Property Operations

### lookup

Find a property entry in an object.

```typescript
Porffor.object.lookup(obj: object, target: any): i32
```

<ParamField path="obj" type="object" required>
  Object to search in
</ParamField>

<ParamField path="target" type="any" required>
  Property key to find
</ParamField>

<ResponseField name="return" type="i32">
  Pointer to property entry, or 0 if not found
</ResponseField>

### get

Get a property value from an object.

```typescript
Porffor.object.get(obj: any, key: any): any
```

<ParamField path="obj" type="any" required>
  Object to get property from
</ParamField>

<ParamField path="key" type="any" required>
  Property key
</ParamField>

<ResponseField name="return" type="any">
  Value of the property
</ResponseField>

### set

Set a property value on an object.

```typescript
Porffor.object.set(obj: object, key: any, value: any): any
```

<ParamField path="obj" type="object" required>
  Object to set property on
</ParamField>

<ParamField path="key" type="any" required>
  Property key
</ParamField>

<ParamField path="value" type="any" required>
  Value to set
</ParamField>

<ResponseField name="return" type="any">
  The value that was set
</ResponseField>

### define

Define a property with specific flags.

```typescript
Porffor.object.define(obj: object, key: any, value: any, flags: i32): void
```

<ParamField path="obj" type="object" required>
  Object to define property on
</ParamField>

<ParamField path="key" type="any" required>
  Property key
</ParamField>

<ParamField path="value" type="any" required>
  Property value
</ParamField>

<ParamField path="flags" type="i32" required>
  Property descriptor flags:
  - `0b0001` - accessor
  - `0b0010` - configurable
  - `0b0100` - enumerable
  - `0b1000` - writable
</ParamField>

### delete

Delete a property from an object.

```typescript
Porffor.object.delete(obj: object, key: any): boolean
```

<ParamField path="obj" type="object" required>
  Object to delete property from
</ParamField>

<ParamField path="key" type="any" required>
  Property key to delete
</ParamField>

<ResponseField name="return" type="boolean">
  `true` if property was deleted, `false` otherwise
</ResponseField>

## Usage Examples

### Dynamic Property Access

```javascript
function safeGet(obj, key, defaultValue) {
  const entry = Porffor.object.lookup(obj, key);
  if (entry === 0) return defaultValue;
  return Porffor.object.get(obj, key);
}
```

### Creating Frozen Objects

```javascript
function createFrozen(data) {
  const obj = {};
  for (const key in data) {
    // Define as non-writable, non-configurable, enumerable
    Porffor.object.define(obj, key, data[key], 0b0100);
  }
  Porffor.object.preventExtensions(obj);
  return obj;
}
```

### Property Enumeration

```javascript
function getEnumerableKeys(obj) {
  const keys = [];
```
